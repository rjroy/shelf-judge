The Redundancy Setting is embedded in the Collection page right now. This does not feel right.

It needs a dedicated page. We have an `Import BGG` and an `Axes` entry on the sidebar. This needs to be moved to `Redundancies` (or how ever you spell that) under that `Settings` section on the sidebar. There also needs to be a save button which also triggers the values to regenerate. Currently it automatically saves but doesn't regenerate. That's a disconnect from how a user thinks. They think "save the settings to update the games" not "first I need to make the settings change which automatically saves, then I need to remember to normalize the fitness to make sure everything is setup".

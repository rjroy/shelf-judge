Responsive behavior notes
Mobile (<900px): Sidebar collapses to a top strip (brand + horizontal nav items). Game hero stacks vertically: cover, info, then score hero as a full-width row. Detail panels stack: breakdown table first, rating form below. Tension panel stacks its two signal cards. Collection table compresses: Game + Score + Confidence columns always visible; BGG/Players/Added columns hidden behind swipe or a "More" expansion. Confidence breakdown popover becomes a full-screen bottom sheet.

Tablet (800–1100px): Sidebar stays but detail panels use a single column. Collection table shows all columns but compressed. Improve grid collapses to 1 column.

New file: rating-labels.ts

  • Export RATING_LABELS: Record<number, string> mapping 1–10 to the spec-defined labels:
    1→Offensive, 2→Inexplicable, 3→Just Bad, 4→Not Good, 5→Fine, 6→Good, 7→Very Good, 8→Recommended, 9→Definitive, 10→Essential
  • Export getRatingLabel(rating: number | null): string | null:
    • Returns null for null input
    • Rounds via Math.round() before lookup
    • Returns null for values outside 1–10 after rounding
